CKEditor in Drupal 8
====================

CKEditor in Drupal 8 - The New Possibilities
(DrupalCon Amsterdam, September 30, 2014)

### Installation

```
git clone --recursive https://github.com/wwalc/ckeditor-in-drupal-8.git
```
Then open slides in a browser.
